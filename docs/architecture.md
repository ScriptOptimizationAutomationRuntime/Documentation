# SOAR Architecture

## Overview

SOAR is a program that can help you with developer tasks.

---

## Execution Flow

1. User inputs a command through the window it is running on
2. Command is processed
3. The appropriate module or script is located  
4. Command is executed  
5. Output and logs are recorded

---

## Design Goals

- Keep the system lightweight and simple
- Maintain modular structure for easy expansion
- Avoid unnecessary complexity found in larger workflow tools
- Make automation scripts easier to manage and reuse

---

## Notes

SOAR architecture is actively evolving and may change as new features and modules are introduced.
