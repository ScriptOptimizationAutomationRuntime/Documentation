# SOAR Overview

## What is SOAR?

SOAR (Script Optimization and Automation Runtime) is a Python-based automation runtime designed to help developers organize, manage, and execute scripts in a structured environment.

---

## Purpose

It focuses on:

- Giving specific data on computer.
- Creating projects
- Autocode
- Simple code fixing.

---

## Key Idea

SOAR is a simple program that can help with everyday coding and developer tasks.

---

## Ecosystem

There are multiple respotories for reasons.

- **Latest-Version** – Where you get SOAR.
- **SOAR Dev Edition** – Program where you can create mods for SOAR
- **SOAR Mod Loader** – Normal SOAR version just with Mods easy to install.
- **SOAR Core** – Documentation and reference material (this repository)

---

## Status

SOAR is currently an experimental project and is actively being developed. Features and structure may change over time.
